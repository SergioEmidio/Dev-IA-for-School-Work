# AI-SYSTEM-PRO

Advanced AI system powered by Gemini Flash.

## Features

- Robust architecture
- Secure backend
- Memory persistence
- Vector memory engine
- Gemini Flash integration
- Gemini-like interface
- Strong error handling

## Run Backend

npm install
npm run dev

## Run AI-Engine

cd AI-Engine
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
